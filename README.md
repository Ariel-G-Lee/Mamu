## 1. 프로젝트 소개

2019 mobile programming android team project

## 3. 주요 기능
#### 회원가입/로그인
- 아이디 중복확인, 비밀번호 패턴 체크
#### 날짜별 메모 작성 및 조회
- youtube api 사용(Url을 통한 동영상 첨부)
- 사진 첨부
- 해시태그 입력
#### 메모 검색
- 입력한 해시태그를 이용한 검색
